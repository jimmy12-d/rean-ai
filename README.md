# Rean AI: Khmer Grade 12 Tutor

Rean AI is a specialized, privacy-focused AI tutor designed to help Cambodian students prepare for the Grade 12 National Exam (BACII). It combines fine-tuned LLMs with a Retrieval-Augmented Generation (RAG) pipeline to provide accurate, curriculum-aligned answers in Physics, History, Biology, and Earth Science.
